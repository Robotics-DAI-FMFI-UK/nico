Forward and inverse kinematics for Nico:

Requirements:

IKpy:  https://github.com/Phylliade/ikpy

pip install ikpy
pip install 'ikpy[plot]'


Usage:

python3 nico-forward-kinematics.py

use keys A,S,D,F,G,H,J and Z,X,C,V,B,N,M to change angles, Q to quit
use mouse (and SHIFT key) to change camera view


python3 nico-inverse-kinematics.py

use keys A,S,D and Z,X,C to change target position, Q to quit
use mouse (and SHIFT key) to change camera view


see PNG images for example screenshots
